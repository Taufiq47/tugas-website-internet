Internet

Internet adalah jaringan global yang memungkinkan perangkat seperti komputer, smartphone, dan server berkomunikasi satu sama lain. Data yang kita kirim dan terima melewati berbagai lapisan teknologi dan protokol untuk mencapai tujuan akhir. Agar lebih mudah dipahami, kita bisa memecah cara kerja internet ke dalam beberapa komponen utama:

Bagaimana internet bekerja?

Internet bekerja dengan menghubungkan banyak komputer menggunakan aturan yang disebut Internet Protocol (IP). Setiap perangkat yang terhubung ke internet punya alamat IP yang unik. Saat Anda mengunjungi situs web, perangkat Anda mengirim permintaan ke server yang menyimpan situs tersebut. Permintaan ini melewati beberapa perangkat di jaringan, seperti router dan gateway, sampai akhirnya mencapai server. Setelah itu, server mengirimkan data (misalnya, halaman web) kembali ke perangkat Anda, dan data tersebut ditampilkan di browser.

HTTP & HTTPS

HTTP (Hypertext Transfer Protocol) adalah protokol yang digunakan untuk mentransfer data antara perangkat Anda (klien) dan server. HTTP bekerja melalui serangkaian permintaan dan respons. Contohnya, ketika Anda memasukkan URL ke browser, browser mengirim permintaan HTTP ke server yang menyimpan situs web tersebut, dan server merespons dengan mengirimkan halaman web.

HTTPS (HTTP Secure) adalah versi aman dari HTTP. HTTPS menggunakan SSL/TLS (Secure Sockets Layer/Transport Layer Security) untuk mengenkripsi data yang ditransfer antara perangkat Anda dan server. Ini mencegah data Anda disadap oleh pihak ketiga saat dalam perjalanan, memberikan keamanan tambahan terutama saat bertransaksi online atau mengakses informasi sensitif.

Apa itu Domain?

Nama domain adalah alamat yang mudah diingat untuk mengakses situs web, seperti www.google.com. Nama domain berfungsi sebagai pengganti alamat IP yang lebih sulit dihafalkan. Ketika Anda mengetikkan nama domain di browser, sistem DNS (Domain Name System) akan menerjemahkan nama domain tersebut menjadi alamat IP yang terkait, memungkinkan perangkat Anda menemukan dan menghubungi server yang benar.
